Using paths to create nested elements

Here is an example of how to use the @Path annotation to create nested XML elements. 


   @Root
   public class Point {
      
      @Element
      private final int x;
      
      @Element
      private final int y;
      
      public Point(@Element(name="x") int x, @Element(name="y") int y) {
         this.x = x;
         this.y = y;
      }
   }
   

   
   <point>
     <x>10</x>
     <y>4</y>
   </point>
