Using paths to create nested elements

Here is an example of how to use the @Path annotation to create nested XML elements. 


   @Root
   public class Example {
      
      @Path("a/b[1]")
      @Element
      private String x;
      
      @Path("a/b[2]")
      @Element
      private String y;
   }
   

   
   <example>
     <a>
       <b>
         <x>foo</x>
       </b>
       <b>
         <y>bar</y>
       </b>
     </a>
   </example>
